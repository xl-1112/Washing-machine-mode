# Washing-machine-mode

e洁小程序「模式统计」功能原型，可发布到 GitHub 和 Tencent Cloud EdgeOne Pages。

## 文件

- `index.html`：已内联 CSS 和 JavaScript，可作为静态站点入口直接托管。

## EdgeOne Pages 配置

如果通过 GitHub 导入：

- Framework preset：None / Static
- Build command：留空
- Output directory：`/`
- Entry file：`index.html`

如果直接上传：

- 上传 `index.html` 或包含 `index.html` 的 zip 包。
